# Silent mode

## What is it?

Silent mode is an unattended installation flow that lets you install the SAP Agent without interacting with the InstallShield wizard screens. It is useful when scripting deployments across many machines or when an administrator cannot be present at install time.

## Where to find the silent mode procedure

The SAP Agent silent mode procedure is shared with all OpCon components. To install the SAP Agent in silent mode, refer to the [Silent Mode](https://help.smatechnologies.com/opcon/core/installation/components#silent-mode) section in the **OpCon Installation** online help.
