# Installation overview

## What is it?

The Installation section walks you through everything required to put the SAP Agent on a Windows host and connect it to OpCon. Start with prerequisites to confirm the SAP system is ready, decide on installation media, run a new install or an upgrade, and add additional instances if you need more than one agent on the same machine.

## In this section

- [Prerequisites](./prerequisites.md) — SAP release level, XBP interface, login account, and authorization roles required before installing.
- [Installation media](./determine-installation-media.md) — Choose the delivery method (USB, FTP, DVD-ROM, or ISO) for the installer.
- [New installation](./new-installation.md) — Run the installer, configure service startup, and create the SAP machine record in OpCon.
- [Upgrade installation](./upgrade-installation.md) — Upgrade an existing install, including 32-bit to 64-bit transitions and the 19.1.0 proxy migration.
- [Silent mode](./silent-mode.md) — Pointer to the shared OpCon silent install procedure for unattended deployments.
- [Multiple instances](./multiple-instances.md) — Install or upgrade more than one SAP Agent instance on the same Windows host.
