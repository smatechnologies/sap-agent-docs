# Administration overview

## What is it?

The Administration section covers day-to-day management of an installed SAP Agent. It documents every setting in the SAPLSAM.ini configuration file and explains how to start and stop the SAP Agent and its companion JORS service from the Windows Service Control Manager.

## In this section

- [Configuration file](./configuration-file.md) — Reference for SAPLSAM.ini, including general, network, debug, SAP system, job tracking, and JORS settings.
- [Managing the agent](./manage-lsam.md) — Start and stop the SAP Agent and JORS services using the Windows Service Control Manager.
