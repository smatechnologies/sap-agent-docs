# Reference overview

## What is it?

The Reference section is the lookup material you reach for when triaging a SAP job: the short machine messages OpCon shows next to job status, the SAP Agent's exit codes for failed jobs, and the SAP BAPI error numbers surfaced through the XBP interface.

## In this section

- [Machine messages](./machine-messages.md) — The 20-character status messages the SAP Agent shows next to OpCon Job Status, including child job status codes.
- [Exit codes](./exit-codes.md) — SAP Agent-specific exit codes for failed jobs and what each one indicates.
- [SAP BAPI errors](./sap-bapi-errors.md) — Reference list of SAP BAPI error message numbers and descriptions surfaced through the XBP interface.
